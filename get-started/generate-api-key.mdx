---
title: 'Generate API key'
description: 'To authenticate and securely send logs'
---

<img className="block w-full rounded-lg" src="/images/api.png" alt="Generate API Key" />

<li>From the home page, hit `C` to create a new API key.</li>
<li>Alternatively, you can go to Settings from the top left corner, or by hitting `G then S`, and navigate to the API section.</li>

<Info>
    Make sure to copy and store API key securely. You will not be able to retrieve it later.
</Info>
